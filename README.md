a# OpenPlate

A food transparency platform where home chefs and small restaurants share their actual cooking process — so customers make ordering decisions based on verified quality, not just photos.

## The Problem
Food delivery apps show marketing photos but nothing about how food is actually prepared. There is zero visibility into hygiene, ingredients, or cooking methods.

## The Solution
OpenPlate lets food providers share step-by-step cooking process posts. Customers can see exactly how their food is made before ordering.

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: SQLite (Prisma ORM) → PostgreSQL in production
- Auth: JWT with refresh tokens

## Project Structure
- `client/` — React frontend
- `server/` — Node.js + Express backend
